# Delhivery — Graph-Based Network Intelligence

> Optimizing Delivery ETAs with Graph ML | Machine Learning Consulting Project

## Problem Statement
Delhivery's OSRM-based ETA system significantly underestimates delivery times on a large fraction of routes. This project builds a graph-based intelligent system to produce more accurate ETAs, surface bottleneck hubs, and generate actionable recommendations for operations leadership.
